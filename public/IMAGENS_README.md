# Imagens Necessárias

Para que o site funcione completamente, você precisa adicionar as seguintes imagens na pasta `public/`:

## Imagens Obrigatórias

1. **hero-image.jpg** - Imagem inspiradora para a Hero Section (recomendado: 1200x800px)
2. **about-image.jpg** - Foto simpática para a seção "Sobre Nós" (recomendado: 800x600px)
3. **ebook-patagonia.jpg** - Capa do e-book "Guia Completo da Patagônia" (recomendado: 400x600px)
4. **ebook-nomade.jpg** - Capa do e-book "Manual do Nômade Digital" (recomendado: 400x600px)

## Imagens do Blog (Destaques)

1. **blog-post-1.jpg** - Para o artigo "10 Dicas Essenciais para Viajar com Orçamento Limitado" (recomendado: 600x400px)
2. **blog-post-2.jpg** - Para o artigo "Como Planejar uma Viagem de 3 Meses pela Europa" (recomendado: 600x400px)
3. **blog-post-3.jpg** - Para o artigo "Os Melhores Destinos para Nômades Digitais em 2024" (recomendado: 600x400px)
4. **blog-post-4.jpg** - Para o artigo "Checklist Completo: O Que Levar em uma Mochila de Viagem" (recomendado: 600x400px)

## Notas

- Todas as imagens devem ser otimizadas para web (formato JPG ou WebP)
- Use imagens de alta qualidade, mas com tamanho de arquivo otimizado
- As imagens serão exibidas de forma responsiva, então proporções próximas às recomendadas são importantes
